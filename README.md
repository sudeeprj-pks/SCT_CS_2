Simple Image Encryption Tool

A beginner-friendly Python project that encrypts and decrypts images using **pixel manipulation** techniques.  
This project demonstrates basic cryptography concepts applied to image files using simple arithmetic operations.



Features
- Encrypts and decrypts image files (JPG, PNG, etc.)
-  Uses simple pixel manipulation (addition/subtraction with a key)
-  Easy to understand for cybersecurity or Python learners
-  Works completely offline
-  Lightweight and dependency-free except for Pillow



 Requirements
 Python 3.x  
 Pillow library (install using pip)
